# Biblioteca de Prompts

Repositório para organizar, versionar e reutilizar prompts de IA.

## Estrutura

- `pesquisa/` — prompts para pesquisa e análise de mercado
- `escrita/` — prompts para textos, relatórios e documentos
- `agentes/` — prompts e instruções para agentes de IA

## Como usar

Cada pasta contém um arquivo `.md` por prompt, com:
- **Objetivo** — para que serve
- **Prompt** — o texto completo
- **Exemplo de output** — resultado esperado
- **Notas** — variações e dicas de uso
